# AI Meta SEO Assistant

A free WordPress plugin that uses ChatGPT to generate meta titles, descriptions, keyword suggestions, and readability analysis.

## Features
- Auto-generate meta title and description using ChatGPT
- Keyword suggestions
- Title/description rewriting in different tones
- SEO score and readability
- Internal link suggestions

## How to Install
1. Upload the plugin to `/wp-content/plugins/`
2. Activate via the WordPress admin panel
3. Set your OpenAI API key in Settings > AI Meta SEO Assistant
